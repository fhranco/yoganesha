# ॐ Yoganesha

**Yoganesha** es un proyecto boutique de Yoga y Bienestar que fusiona la disciplina física con la sabiduría espiritual de Ganesha.

## 🎨 Identidad Visual
- **Colores:** Saffron (#FF9933), Deep Purple (#4A148C), Ivory White (#F9F7F2).
- **Tipografía:** Outfit (Títulos), Inter (Cuerpo).
- **Estética:** Premium, Zen, Moderno, Minimalista.

## 🚀 Estructura del Proyecto
- `index.html`: Estructura semántica SEO-friendly.
- `style.css`: Sistema de diseño basado en variables CSS, flexbox/grid y animaciones fluidas.
- `main.js`: Interactividad ligera y efectos de scroll.
- `assets/`: Recursos visuales del proyecto.
